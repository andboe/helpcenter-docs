= Navigation

* Defined in one or more navigation files
* Regular AsciiDoc list in a regular AsciiDoc file
* Can be managed in a central file or can be assembled from multiple files (e.g. one nav file per module)

.The navigation file for the getting-started module

.Multiple nav files registered in the antora.yml
[,yaml]
----
nav:
  - modules/getting-started/nav.adoc
  - modules/forecast/nav.adoc
  - modules/plan/nav.adoc
  - modules/me/nav.adoc
----

[NOTE]
====
Nav files support all standard AsciiDoc/Antora features: includes, conditionals, formatting.
It's ultra-flexible!
====
